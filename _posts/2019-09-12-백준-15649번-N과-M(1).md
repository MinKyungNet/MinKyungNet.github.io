출제 의도는 이렇게 푸는게 아니었을거다.    
순열을 만들어주는 itertools.permutations를 사용했고,     
형식에 맞게 출력해줬다.

```python
import itertools
N, M = map(int, input().split())
arr = [i for i in range(1, N+1)]
perm = list(itertools.permutations(arr, M))
for i in perm:
    for j in i:
        print(j, end=" ")
    print()

```

다른 사람 코드

```python
def permutation(arr, l):
    arr = sorted(arr)
    use = [0 for _ in range(len(arr))]

    def generate(choice, use):
        if len(choice) == l:
            for i in choice:
                print(i, end=" ")
            print()
            return

        for i in range(len(arr)):
            if not use[i]:
                choice.append(arr[i])
                use[i] = 1
                generate(choice, use)
                choice.pop()
                use[i] = 0
    generate([], use)

N, M = map(int, input().split())
arr = [i for i in range(1, N + 1)]
permutation(arr, M)
```
